\section{Zielsetzung}
Ziel dieser Arbeit ist die Entwicklung und Validierung einer Motorsteuerung für einen \SI{36}{\volt} BLDC-Motor. Der Controller soll für einen Dauerphasenstrom von \SI{30}{\ampere} ausgelegt werden. Als Ansteuerverfahren wird Space Vector PWM (SVPWM) implementiert.

\section{Abgrenzung}
Im Rahmen dieser Arbeit wird bewusst auf eine feldorientierte Regelung (FOC) verzichtet. Stattdessen liegt der Fokus auf einer **Open-Loop SVPWM-Ansteuerung**. 

Diese Entscheidung wurde getroffen, um:
\begin{itemize}
    \item einen robusten und sicheren Anlauf des Motors zu gewährleisten,
    \item die Komplexität der Software (Vermeidung von Park-/Clarke-Transformationen und PID-Reglern) für den ersten Prototypen zu reduzieren,
    \item den Fokus auf das Hardware-Design und die thermische Auslegung zu legen.
\end{itemize}
